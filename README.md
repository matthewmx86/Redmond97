    <article class="markdown-body entry-content p-3 p-md-6" itemprop="text"><h1><a id="user-content-redmond97" class="anchor" aria-hidden="true" href="#redmond97"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Redmond97</h1>
<p>A Win9x inspired theme for GTK3 and GTK2 developed for XFCE4
<a target="_blank" rel="noopener noreferrer" href="https://github.com/matthewmx86/Redmond97/blob/master/Screenshots/Screenshot.png"><img src="https://github.com/matthewmx86/Redmond97/raw/master/Screenshots/Screenshot.png" alt="Image Screenshot" style="max-width:100%;"></a></p>
<h2><a id="user-content-about" class="anchor" aria-hidden="true" href="#about"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>About</h2>
<p>The Redmond97 project aims to recreate the nostalgic look of the Win9x desktop for the XFCE4 desktop environment.
The goal is to provide the clean and professional look of the classic Windows desktop while enjoying
the stability and flexability of Linux. A side effect of the theme is that programs running under wine
also integrate nicely with the Redmond97 theme. The theme should be useful for those who are new to Linux
and would prefer a familiar interface or those feeling a little nostalgic for the classic look.</p>
<h2><a id="user-content-extras" class="anchor" aria-hidden="true" href="#extras"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Extras</h2>
<p>Included with the main theme package are the GTK themes, the Xfce4WM theme, and a Firefox classic IE3 theme.
Many color schemes from the Windows 98 Plus! pack are also included. A theme generator script is also available
to compile the Redmond97 theme using custom colors.</p>
<h2><a id="user-content-requirements" class="anchor" aria-hidden="true" href="#requirements"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Requirements</h2>
<h4><a id="user-content-main-theme" class="anchor" aria-hidden="true" href="#main-theme"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Main theme</h4>
<p>The following packages are recommended for full functionality:</p>
<pre><code>firefox, xfce4, xfce4-goodies, xfce4-whiskermenu-plugin (included with xfce4-goodies), gtk-engines, gtk-nocsd
</code></pre>
<p>The theme has been designed for XFCE4 so the XFCE4 desktop environment is highly recommended but not required
for the use of the GTK and Firefox themes. There is no Window Manager theme support for desktop environments
other than XFCE4 at this time. The theme also includes support for the GTK3 version of Whisker Menu if available.</p>
<h4><a id="user-content-redmond97-theme-builder" class="anchor" aria-hidden="true" href="#redmond97-theme-builder"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Redmond97 Theme Builder</h4>

<pre><code>imagemagick, bc, sed, grep, tar
</code></pre>
<h2><a id="user-content-installation" class="anchor" aria-hidden="true" href="#installation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Installation</h2>
<h4><a id="user-content-main-theme-1" class="anchor" aria-hidden="true" href="#main-theme-1"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Main theme</h4>
<p>Make a .themes directory in your home directory if one doesn't exist and extract the Redmond97.tar.gz archive into
the ~/.themes directory.</p>
<pre><code>mkdir ~/.themes
tar -xvzf Redmond97.tar.gz -C ~/.themes/
</code></pre>
<p>The GTK2/3 and Xfce4WM themes will now be installed.
It is also recommended to disable GTK overlay scrollbars (autohiding scrollbars in GTK3). The following command
will disable the overlay scrollbars for the current user:</p>
<pre><code>export GTK_OVERLAY_SCROLLING=0
</code></pre>
<p>You may have to log out and back in for the setting to take effect.</p>
<h4><a id="user-content-firefox-theme" class="anchor" aria-hidden="true" href="#firefox-theme"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Firefox theme</h4>
<p>You will first need to find your firefox user profile directory. It is usually the one that ends with ".default".
To find the correct directory, open a terminal and go to the hidden Firefox directory. Using grep you can view the directories
ending with ".default".</p>
<pre><code>cd ~/.mozilla/firefox
ls | grep default
</code></pre>
<p>In this exmaple I have two directories: one .default and the other .default-release.
<a target="_blank" rel="noopener noreferrer" href="https://github.com/matthewmx86/Redmond97/blob/master/Screenshots/console.png"><img src="https://github.com/matthewmx86/Redmond97/raw/master/Screenshots/console.png" alt="Image Screenshot" style="max-width:100%;"></a>
If you only have one directory ending with .default that one is the correct profile directory and you can skip
this next step. Otherwise, you can run the following to see which profile is the default.</p>
<pre><code>firefox -P
</code></pre>
<p>You will then see the following window:</p>
<p><a target="_blank" rel="noopener noreferrer" href="https://github.com/matthewmx86/Redmond97/blob/master/Screenshots/firefox.png"><img src="https://github.com/matthewmx86/Redmond97/raw/master/Screenshots/firefox.png" alt="Image Screenshot" style="max-width:100%;"></a></p>
<p>The selected profile is your default profile, in my case it is the default-release profile.</p>
<p>Once you have found the correct profile directory, you will then need to make a directory inside of it called "chrome".
Following my example above you would run the command:</p>
<pre><code>mkdir ~/.mozilla/firefox/vugvl4ul.default-release/chrome
</code></pre>
<p>Now that the chrome directory has been created, you can install the classic IE5 theme by extracting the
ie3_classic_firefox.tar.gz archive into your chrome folder. Again, using my example above the command would be:</p>
<pre><code>tar -xvzf ie3_classic_firefox.tar.gz -C ~/.mozilla/firefox/vugvl4ul.default-release/chrome/
</code></pre>
<p>The Firefox theme should now be installed and will be activated once you close all Firefox sessions and restart Firefox.</p>
